# Tic‑Tac‑Toe (Крестики‑нолики)

Веб‑приложение классической игры «Крестики‑нолики», реализованное на Go в рамках учебного буткемпа в Школе 21.

## Особенности

- Игра для одного игрока
- Графический интерфейс не реализован — взаимодействие через HTTP-запросы с JSON
- Реализация оптимальной игры AI с помощью алгоритма minimax
- Определение победителя и ничьей
- Извлечение информации о состоянии текущих/завершенных игр
- Хранение в памяти с помощью sync.Map
- Многослойная архитектура (сборка с помощью uber/fx)

## Технологии

- **Go** (net/http, шаблонизация)
- **Git** – контроль версий

## Запуск

```bash
git clone https://github.com/JUncouth/tic-tac-toe.git
cd tic-tac-toe/src
go run cmd/server/main.go
```
